movespy¶

movespy simplifies interaction with MOVES [1] for project-level analysis.

Author¶

Eric S. Talbot <ericstalbot@gmail.com>

License¶

License is the BSD 2-clause license. See license.txt.

Installation¶

To install movespy, copy the source files to any importable location. Also edit the “movespy_settings.py” module:

  • moves_dir: a string that is the path to you MOVES installation directory
  • moves_db: a string that is the name of your MOVES database (assumed to be local)

Usage¶

The following code executes a complete MOVES run and retrieves the result:

links = {1: {'grade': -1.2,
             'length': 0.25,
             'opmode_distr': {21: {11: 1.0}},
             'road_type': 3,
             'source_distr': {21: 1.0},
             'speed': 34,
             'volume': 400}}

activity =  {'age_distr': {21: {5: 1.0}},
             'county': 50027,
             'day_type': 5,
             'hour': 16,
             'month': 6,
             'year': 2015,
             'links': links}

options = {'detail': 'average',
           'pollutants': [2]}

import movespy.moves
moves = movespy.moves.Moves(activity, options)
emissions_out, activity_out = moves.run()
total_emissions = sum([row.quantity for row in emissions_out])
[1]MOVES is a software package developed by the U.S. Environmental Protection Agency for estimating air pollution from vehicles. Using MOVES is required for certain federally prescribed analyses. For more information on MOVES, see http://www.epa.gov/otaq/models/moves/index.htm.
